# Inspection Profile Service

A profile management microservice for the InspectPro property inspection platform.
Handles user identities, professional profiles, credential verification and subscription based feature access.

## Tech Stack
- **Language:** Kotlin 2+
- **Framework:** Spring Boot 3 + Spring Web MVC
- **Database:** PostgreSQL 16
- **Cache:** Redis 7
- **Migrations:** Flyway
- **Build:** Maven
- **Infrastructure:** Docker + Docker Compose
